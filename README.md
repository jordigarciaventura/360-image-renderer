# 360 renderer
Blender add-on for render your models up to 360ยบ as image sequence.

## COMPATIBILITY
Tested versions:
- Blender 2.81
- Blender 2.83
- Blender 2.92
- Blender 3.2.1

## INSTALLATION
In GitHub:

1. Download the [latest release](https://github.com/jordigarciaventura/360-renderer/releases/latest) ZIP.

In Blender:

2. Go to `Edit > Preferences > Add-ons > Install...`
3. Select the ZIP.
4. Enable the add-on.
